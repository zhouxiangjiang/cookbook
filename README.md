## Setup

```bash
# Ubuntu 12.04.4
sudo apt-get update
sudo apt-get install python3 git

git config --global user.name 'Your Name'
git config --global user.email your-github-email

git clone https://github.com/leven-cn/cookbook.git
cd cookbook
python3 setup.py
```
